This  provides a crucial bridge between the historical concept of "classful" networking and the modern "classless" subnetting we use today. It explains how to identify address classes by their first octet and, more importantly, how to calculate the four key values for any subnet: the Network Address, First Usable Host, Broadcast Address, and Last Usable Host.



### IP Address Classes and Basic Subnetting Cheat Sheet

This guide covers the legacy class-based system, how to identify a class by its first octet, and the fundamental calculations for determining a subnet's range.

---

#### The Legacy of IP Address Classes (Pre-1993)

While classful networking is no longer used, it provides the starting point for understanding subnet masks. The class of an address implies a default subnet mask.

| Class | Default Subnet Mask | Network / Host Split (Bits) | Description |
| :--- | :--- | :--- | :--- |
| **Class A** | **255.0.0.0** | **8 bits** Network / **24 bits** Host | Very large networks. |
| **Class B** | **255.255.0.0** | **16 bits** Network / **16 bits** Host | Medium-sized networks. |
| **Class C** | **255.255.255.0** | **24 bits** Network / **8 bits** Host | Small networks. |
| **Class D** | (N/A - Multicast) | N/A | Used for multicast groups, not assigned to hosts. |
| **Class E** | (N/A - Reserved) | N/A | Reserved for experimental or future use. |

---

#### Identifying the Class by the First Octet

You can instantly tell the class of an address by looking at its first decimal number.

| Class | First Octet (Decimal Range) | First Octet (Binary Pattern) |
| :--- | :--- | :--- |
| **Class A** | **0 - 127** | **0** _ _ _ _ _ _ _ (Starts with 0) |
| **Class B** | **128 - 191** | **10** _ _ _ _ _ _ (Starts with 10) |
| **Class C** | **192 - 223** | **110** _ _ _ _ _ (Starts with 110) |
| **Class D** | **224 - 239** | **1110** _ _ _ _ (Starts with 1110) |
| **Class E** | **240 - 255** | **1111** _ _ _ _ (Starts with 1111) |

---

#### The Four Essential Subnet Values

For any given IP address and its subnet mask, these four values define the boundaries of that network.

| Value | How to Calculate | Purpose |
| :--- | :--- | :--- |
| **1. Network Address** | Set **all host bits to 0**. | Identifies the network itself. Cannot be assigned to a host. Used in routing tables. |
| **2. First Usable Host** | **Network Address + 1**. | The first IP address that can be assigned to a device on this network. |
| **3. Broadcast Address** | Set **all host bits to 1**. | A special address used to send traffic to **all** hosts on this local network. Cannot be assigned to a host. |
| **4. Last Usable Host** | **Broadcast Address - 1**. | The last IP address that can be assigned to a device on this network. |

---

#### Worked Examples (Using Default Classful Masks)

**Example 1: Class A (10.74.222.11 / 255.0.0.0)**

| Calculation Step | Network (1st Octet) | Host (Last 3 Octets) | Resulting IP |
| :--- | :--- | :--- | :--- |
| Given IP | **10** | 74 . 222 . 11 | `10.74.222.11` |
| **1. Network Address** | **10** | **0 . 0 . 0** (Host bits to 0) | **`10.0.0.0`** |
| **2. First Usable Host** | **10** | **0 . 0 . 1** (Network + 1) | **`10.0.0.1`** |
| **3. Broadcast Address** | **10** | **255 . 255 . 255** (Host bits to 1) | **`10.255.255.255`** |
| **4. Last Usable Host** | **10** | **255 . 255 . 254** (Broadcast - 1) | **`10.255.255.254`** |

**Example 2: Class B (172.16.88.200 / 255.255.0.0)**

| Calculation Step | Network (First 2 Octets) | Host (Last 2 Octets) | Resulting IP |
| :--- | :--- | :--- | :--- |
| Given IP | **172.16** | 88 . 200 | `172.16.88.200` |
| **1. Network Address** | **172.16** | **0 . 0** | **`172.16.0.0`** |
| **2. First Usable Host** | **172.16** | **0 . 1** | **`172.16.0.1`** |
| **3. Broadcast Address** | **172.16** | **255 . 255** | **`172.16.255.255`** |
| **4. Last Usable Host** | **172.16** | **255 . 254** | **`172.16.255.254`** |

**Example 3: Class C (192.168.4.77 / 255.255.255.0)**

| Calculation Step | Network (First 3 Octets) | Host (Last Octet) | Resulting IP |
| :--- | :--- | :--- | :--- |
| Given IP | **192.168.4** | 77 | `192.168.4.77` |
| **1. Network Address** | **192.168.4** | **0** | **`192.168.4.0`** |
| **2. First Usable Host** | **192.168.4** | **1** | **`192.168.4.1`** |
| **3. Broadcast Address** | **192.168.4** | **255** | **`192.168.4.255`** |
| **4. Last Usable Host** | **192.168.4** | **254** | **`192.168.4.254`** |

This process of identifying the network/host boundary and calculating these four values is the exact same method used for classless subnetting, which we will cover next.
