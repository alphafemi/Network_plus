### Ethernet Standards Cheat Sheet

This guide covers the basics of Ethernet, the IEEE 802.3 standards, and how to interpret common Ethernet naming conventions.

---

#### The Basics of Ethernet

| Concept | Description |
| :--- | :--- |
| **What is Ethernet?** | The most popular and widely used family of technologies for wired local area networks (LANs). It defines the wiring, signaling, and frame structures used for communication. |
| **Governing Body** | **IEEE 802.3 Committee.** This group within the Institute of Electrical and Electronics Engineers is responsible for creating and maintaining the standards that ensure all Ethernet devices can work together. |
| **Why it's successful** | A single, open set of standards ensures interoperability. Any device from any manufacturer that conforms to the standard can connect to any Ethernet network. |

---

#### Decoding Ethernet Standard Names

The naming convention for Ethernet standards, while not a strict technical rulebook, provides useful clues about the technology.

**The Format:** **[Speed]** **BASE** **-[Media Indicator]**

Let's break down the example **`1000BASE-T`** :

| Part | Meaning | Description |
| :--- | :--- | :--- |
| **1000** | **Speed** | Indicates the data rate of the standard. In this case, **1000 Mbps** (1 Gigabit per second). <br> *Other examples: `10BASE-T` (10 Mbps), `10GBASE-T` (10 Gbps).* |
| **BASE** | **Baseband** | Refers to **baseband** signaling. This means the entire bandwidth of the cable is used for a single signal (one channel). This is in contrast to **broadband**, which uses multiple frequencies to carry multiple channels (like cable TV). |
| **-T** | **Media Type** | Indicates the type of cabling used. <br> - **`-T`** : **Twisted-pair copper** cabling (like Cat5e, Cat6). <br> - **`-F`** : **Fiber optic** cabling. <br> - **`-SX`** : Fiber optic, specifically for **short wavelength** (multimode fiber). <br> - **`-LX`** : Fiber optic, specifically for **long wavelength** (single-mode fiber). |

#### Common Ethernet Standards Examples

| Standard Name | Common Name / Speed | Media Type | Typical Cable |
| :--- | :--- | :--- | :--- |
| **10BASE-T** | 10 Mbps Ethernet | Twisted-Pair Copper | Cat3 or better |
| **100BASE-TX** | Fast Ethernet (100 Mbps) | Twisted-Pair Copper | Cat5 or better |
| **1000BASE-T** | **Gigabit Ethernet (1 Gbps)** | Twisted-Pair Copper | Cat5e or better |
| **10GBASE-T** | 10 Gigabit Ethernet (10 Gbps) | Twisted-Pair Copper | Cat6a or better |
| **1000BASE-SX** | Gigabit Ethernet (1 Gbps) | Fiber Optic (Multimode, short wavelength) | Multimode fiber |
| **1000BASE-LX** | Gigabit Ethernet (1 Gbps) | Fiber Optic (Singlemode/Multimode, long wavelength) | Singlemode fiber (primarily) |
