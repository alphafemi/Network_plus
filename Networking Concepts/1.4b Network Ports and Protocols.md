This  is a practical guide to the most common network ports you'll encounter as an IT professional. Knowing these ports is essential for configuring firewalls, troubleshooting connectivity, and understanding network traffic.



### Common Network Ports Cheat Sheet

This guide lists the application, protocol, port number, and a brief description for each service.

---

#### Remote Access & Terminal Services

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **SSH / SFTP** | TCP | **22** | **Secure Shell.** Used for secure, encrypted remote terminal access and command execution. **SFTP (Secure FTP)** also uses this port for encrypted file transfers. |
| **Telnet** | TCP | **23** | **Telecommunication Network.** Used for remote terminal access. **All data, including passwords, is sent in cleartext.** Largely replaced by SSH for security reasons. |
| **RDP** | TCP | **3389** | **Remote Desktop Protocol.** Used by Microsoft Windows to provide remote graphical desktop access to a computer. |

---

#### File Transfer & Sharing

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **FTP (Data)** | TCP | **20** | **File Transfer Protocol (Data Channel).** Used for the actual file data transfer. |
| **FTP (Control)** | TCP | **21** | **File Transfer Protocol (Control Channel).** Used for sending commands (login, directory listing, etc.). |
| **TFTP** | UDP | **69** | **Trivial File Transfer Protocol.** A simple, unauthenticated file transfer protocol. Often used by network devices (like IP phones) to download configuration files. |
| **SMB / CIFS** | TCP | **445** | **Server Message Block / Common Internet File System.** Used by Microsoft Windows for file and printer sharing, as well as authentication. (Modern NetBIOS-less communication). |

---

#### Email Services

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **SMTP** | TCP | **25** | **Simple Mail Transfer Protocol.** Used for email transfer **between mail servers**. Traditionally unencrypted. |
| **SMTP with TLS** | TCP | **587** | **Simple Mail Transfer Protocol with TLS encryption.** Used by email clients to **submit** outgoing email to a server securely. |
| **IMAP4** | TCP | **143** | **Internet Message Access Protocol.** Used by email clients to **receive** and manage emails stored on a mail server. |
| **POP3** | TCP | **110** | **Post Office Protocol v3.** Used by email clients to **download** emails from a mail server to a local client. |

---

#### Web & Directory Services

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **HTTP** | TCP | **80** | **Hypertext Transfer Protocol.** The foundation of data communication for the web. All data is sent in cleartext. |
| **HTTPS** | TCP | **443** | **Hypertext Transfer Protocol Secure.** HTTP over SSL/TLS encryption. The standard for secure web browsing. |
| **DNS** | UDP / TCP | **53** | **Domain Name System.** Translates domain names to IP addresses. **UDP** is used for standard queries. **TCP** is used for large transfers (like zone transfers between DNS servers). |
| **DHCP** | UDP | **67, 68** | **Dynamic Host Configuration Protocol.** Automatically assigns IP addresses and other network configuration to devices. Port 67 is for the server, port 68 is for the client. |
| **LDAP** | TCP | **389** | **Lightweight Directory Access Protocol.** Used to access and query directory services (like Microsoft Active Directory). |
| **LDAPS** | TCP | **636** | **LDAP Secure.** LDAP over SSL/TLS, providing encrypted communication. |

---

#### Network Management & Logging

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **NTP** | UDP | **123** | **Network Time Protocol.** Synchronizes clocks between computers and network devices over a network. |
| **SNMP (Query)** | UDP | **161** | **Simple Network Management Protocol.** Used by management stations to query network devices for information (statistics, status). |
| **SNMP (Trap)** | UDP | **162** | **Simple Network Management Protocol (Trap).** Used by managed devices to **asynchronously send alerts** or notifications to a management station. |
| **Syslog** | UDP | **514** | **System Logging Protocol.** Used to send log messages and events from network devices to a central logging server (often a SIEM). |

---

#### Databases & Communication

| Application / Service | Protocol | Port Number | Description |
| :--- | :--- | :--- | :--- |
| **MS-SQL** | TCP | **1433** | **Microsoft SQL Server.** The default port for listening to client connections to a Microsoft SQL database. |
| **SIP** | TCP | **5060, 5061** | **Session Initiation Protocol.** Used to control communication sessions in VoIP (Voice over IP), such as setting up and tearing down phone calls. Port 5060 is often unencrypted, 5061 for TLS. |

This list represents the core set of ports you'll need to know for the Network+ exam and for real-world network administration.
